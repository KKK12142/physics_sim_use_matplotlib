# 수업 설계 규칙

## 1. 수업지도안 구성 요소

### 필수 포함 항목

```markdown
# 수업지도안: {단원명}

## 기본 정보
| 항목 | 내용 |
|------|------|
| 과목 | 물리학 I/II |
| 단원 | {단원명} |
| 차시 | {n}/{전체}차시 |
| 학습목표 | {목표} |
| 성취기준 | {코드} |

## 교수학습 과정

### 도입 (10분)
- 전시학습 확인
- 동기유발
- 학습목표 제시

### 전개 (30분)
- 핵심개념 설명
- 예시/시연
- 학생 활동

### 정리 (10분)
- 핵심내용 요약
- 형성평가
- 차시예고
```

## 2. 시간 배분 규칙

### 50분 수업 기준

| 단계 | 시간 | 비율 |
|------|------|------|
| 도입 | 5~10분 | 10~20% |
| 전개 | 30~35분 | 60~70% |
| 정리 | 5~10분 | 10~20% |

### 학생 활동 시간

| 수업 유형 | 학생 활동 비율 |
|----------|---------------|
| 강의 중심 | 20~30% |
| 일반 수업 | 30~40% |
| 탐구 수업 | 50~60% |
| 실험 수업 | 60~70% |

---

## 3. 학습목표 작성 규칙

### ABCD 형식

- **A**udience: 학습자 (생략 가능)
- **B**ehavior: 행동 동사 (설명할 수 있다, 계산할 수 있다)
- **C**ondition: 조건 (주어진 상황에서)
- **D**egree: 수준 (정확하게, 80% 이상)

### 행동 동사 예시

| 수준 | 행동 동사 |
|------|----------|
| 지식 | 정의한다, 나열한다, 명명한다 |
| 이해 | 설명한다, 해석한다, 비교한다 |
| 적용 | 계산한다, 적용한다, 해결한다 |
| 분석 | 분석한다, 구별한다, 추론한다 |
| 평가 | 평가한다, 판단한다, 비판한다 |
| 창안 | 설계한다, 구성한다, 창안한다 |

## 4. 물리 개념 설명 규칙

### 개념 도입 순서

```
1. 일상 경험/현상 제시
2. 과학적 질문 유도
3. 핵심 개념 정의
4. 수학적 표현
5. 예제 적용
6. 확장/심화
```

### 수식 설명 원칙

```
1. 문자 의미 먼저 설명
2. 단위 확인
3. 특수한 경우 검토 (극한값)
4. 그래프로 시각화
5. 실생활 수치 대입
```

### 오개념 주의

모든 수업에서 해당 단원의 대표적 오개념을 명시합니다:

```markdown
## ⚠️ 주의할 오개념

| 오개념 | 바른 개념 |
|--------|----------|
| 무거운 물체가 더 빨리 떨어진다 | 공기저항 무시 시 동시 낙하 |
| 힘이 있어야 운동한다 | 힘이 없으면 등속운동 |
```

## 5. 학생 수준별 차별화

### 수준별 질문

| 수준 | 질문 유형 |
|------|----------|
| 기초 | 개념 확인, 용어 정의 |
| 보통 | 개념 적용, 단순 계산 |
| 심화 | 복합 문제, 실생활 적용 |

### 수준별 활동

```markdown
## 모둠 활동 (수준별)

**기초 과제**: 개념 정리 노트 완성
**보통 과제**: 예제 문제 풀이
**심화 과제**: 실생활 적용 사례 분석
```

## 6. 시각자료 활용 규칙

### PPT 슬라이드 원칙

```
- 한 슬라이드에 하나의 핵심 개념
- 텍스트는 6줄 이내
- 이미지/그래프 필수 포함
- 애니메이션은 개념 이해에 도움될 때만
```

### 시뮬레이션 활용

```
- PhET: 기본 개념 시각화
- Algodoo: 운동 시뮬레이션
- GeoGebra: 수학적 관계 시각화
- Desmos: 그래프 분석
```

## 7. 형성평가 규칙

### 수업 중 확인

```
- 5~10분 간격으로 이해도 확인
- 손들기, 미니화이트보드, 카훗 활용
- 즉각적 피드백 제공
```

### 형성평가 문항 수

| 차시 | 문항 수 | 유형 |
|------|--------|------|
| 1차시 | 3~5문항 | 개념 확인 |
| 2차시 | 5~7문항 | 개념 + 적용 |
| 단원 정리 | 10문항 | 종합 |

## 8. 안전 지도 (실험 수업)

실험 수업 시 반드시 포함:

```markdown
## 🔬 안전 지도

### 실험 전
- 안전 수칙 안내
- 보호 장구 착용 확인
- 실험 기구 점검

### 실험 중
- 교사 순회 지도
- 위험 행동 즉시 제지
- 응급 상황 대비

### 실험 후
- 기구 정리 및 반납
- 폐기물 처리
- 손 씻기
```
