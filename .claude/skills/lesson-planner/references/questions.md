## 카테고리 B: 학생 분석 (Q5~Q8)

### Q5. 학생 수준
```
학생들의 전반적인 수준은 어떤가요?

1. 상위권 중심 (심화 내용 가능)
2. 중위권 중심 (기본 개념 충실)
3. 하위권 포함 (기초부터 설명 필요)
4. 수준 혼합 (수준별 활동 필요)
```

### Q6. 선수학습 상태
```
이 단원에 필요한 선수학습은 되어 있나요?

관련 선수학습:
- 수학: 벡터, 미적분, 삼각함수
- 물리: 이전 단원 개념

1. 충분히 되어 있음
2. 일부 복습 필요
3. 상당 부분 재학습 필요
4. 잘 모르겠음 (진단평가 권장)
```

### Q7. 학생 관심사
```
학생들이 특히 관심 있어하는 분야가 있나요?

3. 진로 연계 (공대, 의대, 교대)
4. 특별한 관심사 없음
5. 기타 (직접 입력)
```

### Q8. 학생 수
```

1. 소규모 (20명 이하)
```

### Q14. 수업 목표 성취기준
```
이 수업의 핵심 성취기준은 무엇인가요?

[2022 개정 교육과정 성취기준 예시]

<물리학 - 힘과 에너지>
[12물리01-01] 물체에 작용하는 알짜힘과 돌림힘이 0일 때 평형을 이룸을 알고, 다양한 구조물의 안정성을 분석할 수 있다.
[12물리01-02] 뉴턴 운동 법칙으로 등가속도 운동을 설명하고 교통안전 사고 예방에 적용할 수 있다.
[12물리01-04] 일과 운동 에너지의 관계를 이해하고, 위치 에너지와 역학적 에너지 보존 법칙을 설명할 수 있다.
[12물리01-06] 열이 역학적 에너지로 전환되는 과정의 효율을 정성적으로 이해하고, 영구기관이 불가능함을 사례를 통해 논증할 수 있다.

<물리학 - 전기와 자기>
[12물리02-01] 전기장과 자기장의 발생 원인을 이해하고, 일상생활에서 전기장과 자기장을 활용한 예를 찾을 수 있다.

<물리학 - 빛과 물질>
[12물리03-01] 빛의 굴절 현상을 이해하고 볼록 렌즈에 의한 상의 형성을 설명할 수 있다.
[12물리03-03] 광전효과와 물질파를 이해하고, 빛과 물질의 이중성을 설명할 수 있다.

<역학과 에너지>
[12역학01-01] 힘의 합성과 분해를 이해하고 여러 힘이 작용하는 물체의 운동을 분석할 수 있다.
[12역학02-01] 열역학 제1법칙을 이해하고 열기관의 열효율을 계산할 수 있다.

<전자기와 양자>
[12전자01-01] 전하 주위의 전기장을 정량적으로 구하고, 전기력선과 등전위면으로부터 전기장의 세기와 방향을 추리할 수 있다.
[12전자03-02] 중첩과 측정을 통한 확률적 상태변화를 이해하고, 양자컴퓨터, 양자암호통신 등의 양자 기술이 미래 사회에 미칠 영향을 인식할 수 있다.

1. 성취기준 코드 입력
2. 핵심 내용으로 설명
3. 잘 모르겠음 (추천해 주세요)
```
