# Concept Explainer: 물리 개념 설명 스킬

## 역할

물리 개념을 학생 수준과 목적에 맞게 다양한 방식으로 설명합니다.
교사가 수업에서 활용하거나 학생이 직접 학습할 때 사용할 수 있습니다.

---

## 설명 수준

### Level 1: 일상 수준 (비전공자/입문)
- 일상 언어 사용
- 비유와 예시 중심
- 수식 최소화

### Level 2: 고등학교 수준 (물리학 I/II)
- 기본 수식 포함
- 교과서 수준 설명
- 문제 풀이 예시

### Level 3: 심화 수준 (경시/과학고)
- 수학적 유도 포함
- 깊이 있는 분석
- 대학 물리 연계

### Level 4: 역사적 맥락
- 과학사적 배경
- 과학자 이야기
- 개념 발전 과정

## 설명 템플릿: 예시

### 운동량 보존 법칙 (Level 2: 고등학교 수준)

```markdown
# 운동량 보존 법칙

## 🎯 핵심 정의

> 외력이 작용하지 않는 고립계에서 충돌 전후 총 운동량은 일정하다.

$$
m_1v_1 + m_2v_2 = m_1v_1' + m_2v_2'
$$

## 🏠 일상 비유

**당구 게임**: 흰 공이 빨간 공을 치면, 흰 공은 느려지고 빨간 공은 빨라집니다.
하지만 두 공의 "움직임의 총량"은 변하지 않습니다.
이것이 바로 운동량 보존입니다!

## 📐 수학적 표현

### 운동량 정의
$$
\vec{p} = m\vec{v}
$$
- 단위: kg·m/s
- 벡터량 (방향 중요!)

### 보존 법칙
$$
\sum \vec{p}_{before} = \sum \vec{p}_{after}
$$

**조건**: 외력 = 0 (고립계)

## 🔬 실험적 증거

### 에어트랙 충돌 실험
- 마찰 최소화
- 충돌 전후 속도 측정
- 운동량 보존 확인

### 측정 결과 예시
| | 충돌 전 | 충돌 후 |
|--|--------|--------|
| 물체1 운동량 | 2.0 kg·m/s | 0.5 kg·m/s |
| 물체2 운동량 | 0 kg·m/s | 1.5 kg·m/s |
| **총 운동량** | **2.0 kg·m/s** | **2.0 kg·m/s** |

## ⚠️ 흔한 오개념

| 오개념 | 바른 개념 |
|--------|----------|
| "운동량은 항상 보존된다" | 외력이 없는 고립계에서만 보존 |
| "속력이 보존된다" | 속력이 아닌 운동량(질량×속도)이 보존 |
| "에너지도 항상 보존된다" | 비탄성 충돌에서 운동에너지는 감소 |

## 🔗 연결 개념

```
운동량 보존
    ├── 충격량-운동량 정리 (F·Δt = Δp)
    ├── 에너지 보존 (탄성 충돌)
    ├── 반발 계수 (e = 분리속도/접근속도)
    └── 로켓 추진 원리
```

## ❓ 확인 질문

1. 질량 2kg인 물체가 3m/s로, 질량 1kg인 물체가 정지해 있다.
   충돌 후 2kg 물체가 1m/s가 되었다면, 1kg 물체의 속도는?

2. 스케이트장에서 서로 밀면 둘 다 반대로 움직이는 이유는?

3. 운동량은 보존되는데 에너지가 보존되지 않는 충돌의 예시는?
```

## 오개념 데이터베이스

### 역학 오개념

```markdown
## 힘과 운동 오개념

### 오개념 1: "힘이 있어야 움직인다"
**증상**: 등속운동 중인 물체에 힘이 작용한다고 생각
**교정**: 힘은 운동 상태의 '변화'를 일으킴. 등속운동은 힘 = 0

### 오개념 2: "무거운 물체가 빨리 떨어진다"
**증상**: 갈릴레이 이전의 아리스토텔레스적 사고
**교정**: 공기저항 무시 시 모든 물체 동시 낙하 (피사의 사탑 실험)

### 오개념 3: "작용-반작용은 상쇄된다"
**증상**: 작용-반작용이 같은 물체에 작용한다고 생각
**교정**: 작용-반작용은 서로 다른 물체에 작용
```

### 전자기 오개념

```markdown
## 전기/자기 오개념

### 오개념 1: "전류가 저항에서 소모된다"
**증상**: 전류가 저항 통과 후 줄어든다고 생각
**교정**: 전류(전하의 흐름)는 일정, 에너지가 소모됨

### 오개념 2: "전지가 전류를 만든다"
**증상**: 전지에서 전류가 나온다고 생각
**교정**: 전지는 전위차를 제공, 전류는 회로 전체에서 동시에 흐름

### 오개념 3: "자석은 모든 금속을 끌어당긴다"
**증상**: 구리, 알루미늄도 자석에 붙는다고 생각
**교정**: 강자성체(철, 니켈, 코발트)만 영구자석에 끌림
```
