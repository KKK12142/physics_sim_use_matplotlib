# 물리 오개념 데이터베이스

## 역학 (Mechanics)

### 힘과 운동

#### MC-01: 힘이 있어야 움직인다
```yaml
오개념: "물체가 움직이려면 힘이 필요하다"
정답: "힘은 운동 상태의 '변화'를 일으킨다. 등속운동은 알짜힘=0"
원인: 일상 경험 (마찰력으로 멈춤), 아리스토텔레스적 사고
교정:
  - 에어트랙/드라이아이스 실험 (마찰 최소화)
  - 우주선 예시 (관성 운동)
수준: 고1-고2
관련단원: 뉴턴 제1법칙
```

#### MC-02: 무거운 물체가 빨리 떨어진다
```yaml
오개념: "무거운 물체가 가벼운 물체보다 빨리 떨어진다"
정답: "공기저항 무시 시 모든 물체는 동일한 가속도 g로 떨어진다"
원인: 공기저항이 있는 일상 경험 (깃털 vs 돌)
교정:
  - 피사의 사탑 사고실험
  - 진공 낙하 실험 영상 (깃털과 볼링공)
  - 달 표면 아폴로 실험
수준: 고1
관련단원: 자유낙하
```

#### MC-03: 작용-반작용은 상쇄된다
```yaml
오개념: "작용과 반작용이 같으니 힘이 상쇄되어 움직이지 않아야 한다"
정답: "작용-반작용은 서로 다른 물체에 작용하므로 상쇄 불가"
원인: 힘의 평형과 혼동
교정:
  - 힘의 평형: 한 물체에 작용하는 여러 힘
  - 작용-반작용: 두 물체 간 상호작용
  - 줄다리기 예시 (지면 마찰력이 결정)
수준: 고1-고2
관련단원: 뉴턴 제3법칙
```

#### MC-04: 등속원운동은 힘이 필요 없다
```yaml
오개념: "등속원운동은 속력이 일정하니까 힘이 없다"
정답: "속력은 일정하지만 방향이 변하므로 구심력이 필요"
원인: 속력과 속도의 혼동
교정:
  - 속력(스칼라) vs 속도(벡터) 구분
  - 회전하는 공 놓으면 접선 방향으로 날아감
수준: 고2
관련단원: 원운동
```

#### MC-05: 위로 던진 물체는 최고점에서 힘=0
```yaml
오개념: "최고점에서 속도가 0이니까 힘도 0이다"
정답: "최고점에서도 중력(mg)이 아래로 작용"
원인: 속도=0과 힘=0 혼동
교정:
  - 속도와 가속도의 독립성
  - 최고점에서 가속도는 여전히 g
수준: 고1
관련단원: 포물선 운동
```

### 에너지

#### MC-06: 에너지는 항상 보존된다 (충돌)
```yaml
오개념: "운동량이 보존되니까 에너지도 보존된다"
정답: "비탄성 충돌에서 운동에너지는 열/소리 등으로 전환됨"
원인: 보존법칙들의 혼동
교정:
  - 탄성 vs 비탄성 충돌 구분
  - 반발계수 개념 도입
  - 실제 충돌 실험 (운동에너지 감소 확인)
수준: 고2
관련단원: 충돌, 에너지 보존
```

#### MC-07: 일과 에너지는 같은 것이다
```yaml
오개념: "일=에너지이다"
정답: "일은 에너지의 전달/변환 방법, 에너지는 상태량"
원인: 단위가 같음 (J)
교정:
  - 일: 과정량 (얼마나 전달했는가)
  - 에너지: 상태량 (얼마나 가지고 있는가)
수준: 고2
관련단원: 일-에너지 정리
```

## 파동 (Waves)

### 파동의 기본

#### WV-01: 파동이 매질을 운반한다
```yaml
오개념: "파동이 지나가면 물도 함께 이동한다"
정답: "파동은 에너지를 전달하지만 매질 자체는 제자리 진동"
원인: 파도 관찰 시 물이 밀려오는 것처럼 보임
교정:
  - 줄파동 시연 (줄은 이동 안 함)
  - 물에 띄운 병뚜껑 관찰 (상하 진동만)
수준: 고1
관련단원: 파동의 특성
```

#### WV-02: 진동수가 높으면 속력이 빠르다
```yaml
오개념: "진동수를 높이면 파동 속력이 빨라진다"
정답: "파동 속력은 매질에 의해 결정됨. v = fλ에서 f↑이면 λ↓"
원인: v = fλ 공식 오해
교정:
  - 기타줄 장력 일정: 진동수↑ → 파장↓, 속력 일정
  - 소리: 같은 공기에서 높은음/낮은음 속력 동일
수준: 고2
관련단원: 파동의 특성
```

### 빛

#### WV-03: 빛은 속도가 줄면 파장이 짧아진다
```yaml
오개념: "매질에서 빛 속력이 느려지면 파장은 변하지 않는다"
정답: "진동수는 일정, 속력↓ → 파장↓ (v = fλ)"
원인: 진동수 불변 개념 누락
교정:
  - 파동의 연속성: 경계면에서 진동수 유지
  - 굴절 현상과 연계
수준: 고2
관련단원: 굴절
```

#### WV-04: 전반사는 모든 입사각에서 일어난다
```yaml
오개념: "밀한 매질에서 소한 매질로 가면 항상 전반사"
정답: "입사각 > 임계각일 때만 전반사 발생"
원인: 전반사 조건 이해 부족
교정:
  - 임계각 공식: sin θc = n₂/n₁
  - 다양한 입사각에서 실험
수준: 고2
관련단원: 전반사
```

## 전자기 (Electromagnetism)

### 전기

#### EM-01: 전류가 저항에서 소모된다
```yaml
오개념: "전류가 저항을 통과하면 줄어든다"
정답: "전류(전하의 흐름)는 일정, 에너지가 소모됨"
원인: 전류와 에너지의 혼동
교정:
  - 직렬회로에서 전류 측정 (어디서나 동일)
  - 저항에서 소모되는 것은 전기에너지 (P = I²R)
수준: 고1-고2
관련단원: 전류, 저항
```

#### EM-02: 전압은 전류에 비례한다
```yaml
오개념: "전압을 높이면 항상 전류가 증가한다"
정답: "옴의 법칙 V=IR에서 R이 일정할 때만 성립"
원인: 옴의 법칙 단순 암기
교정:
  - 저항 변화 시 전류 예측
  - 비옴성 물질 (다이오드) 소개
수준: 고2
관련단원: 옴의 법칙
```

#### EM-03: 전지에서 전류가 나온다
```yaml
오개념: "전류는 전지 +극에서 나와서 -극으로 들어간다"
정답: "전지는 전위차를 유지, 전류는 회로 전체에서 동시에 흐름"
원인: 물 흐름 비유의 한계
교정:
  - 전하는 회로에 이미 존재
  - 전지 = 펌프 (물을 만들지 않고 순환시킴)
수준: 고1
관련단원: 전류, 전압
```

### 볼록렌즈 (물리학 신규)

#### MC-NEW-04: 렌즈 공식 적용 오류
```yaml
오개념: "물체가 초점 안쪽에 있어도 실상이 생긴다"
정답: "초점 안쪽(a < f)에 물체가 있으면 허상(정립확대상)이 생김"
원인: 렌즈 공식 해석 오류
교정:
  - a > 2f: 실상, 도립, 축소
  - f < a < 2f: 실상, 도립, 확대
  - a < f: 허상, 정립, 확대 (돋보기 원리)
수준: 고2
관련단원: 물리학 - 빛과 물질
```
