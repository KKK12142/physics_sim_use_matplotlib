# Experiment Designer: 물리 실험 설계 스킬

## 워크플로우

### Step 1: 실험 선택

```
어떤 실험을 설계할까요?

1. 실험 라이브러리에서 선택
2. 새로운 실험 설계
3. 기존 실험 수정/개선
```

### Step 2: 상세 정보 입력

```
실험 설계에 필요한 정보를 입력해주세요:

1. 대상 학년:
2. 수업 시간 (분):
3. 모둠 구성 (몇 명씩):
4. 사용 가능 기자재:
5. 평가 연계 여부:
```

### Step 3: 문서 생성

선택한 실험에 맞는 모든 문서를 자동 생성합니다.

### Step 4: 검토 및 수정

안전 점검 체크리스트와 함께 최종 검토합니다.

## 안전 지침 라이브러리

### 일반 안전 수칙

```markdown
## 🛡️ 물리 실험실 일반 안전 수칙

### 실험 전
- [ ] 실험복/보호 장갑 착용
- [ ] 긴 머리 묶기
- [ ] 실험 기구 상태 점검
- [ ] 안전 장비 위치 확인 (소화기, 응급함)

### 실험 중
- [ ] 교사 지시에 따라 진행
- [ ] 허가 없이 실험 변경 금지
- [ ] 기구는 조심스럽게 다루기
- [ ] 이상 발생 시 즉시 보고

### 실험 후
- [ ] 기구 정리 및 반납
- [ ] 실험대 정리
- [ ] 손 씻기
```

### 실험별 특수 안전 지침

```markdown
## ⚡ 전기 실험 안전 지침

- 젖은 손으로 전기 기구 만지지 않기
- 회로 연결 전 전원 차단 확인
- 단락(short circuit) 주의
- 고전압 실험 시 절연 장갑 착용

## 🔥 열 실험 안전 지침

- 가열 기구 주변 가연물 제거
- 가열 중인 물체 맨손 접촉 금지
- 화상 발생 시 즉시 찬물로 냉각
- 알코올 램프 사용 시 주의

## 🔬 광학 실험 안전 지침

- 레이저 빔 직접 보지 않기
- 반사광 주의
- 레이저 포인터 조작 시 방향 확인
```

## 데이터 분석 가이드

### 그래프 작성 규칙

```markdown
## 📊 과학적 그래프 작성법

1. **축 설정**
   - 독립변인 → x축
   - 종속변인 → y축
   - 축 제목 + 단위 표기

2. **눈금 설정**
   - 균일한 간격
   - 적절한 범위 (데이터의 80% 이상 포함)

3. **데이터 표시**
   - 점(●)으로 표시
   - 오차 막대 포함 (가능시)

4. **추세선**
   - 최적 적합선 (best fit line)
   - 원점 통과 여부 판단

5. **기울기/절편**
   - 물리적 의미 해석
   - 단위 포함
```

### 오차 분석

```markdown
## 📐 오차 분석 방법

### 오차의 종류

| 종류 | 원인 | 대책 |
|------|------|------|
| 계통 오차 | 기구 결함, 측정 방법 | 보정, 방법 개선 |
| 우연 오차 | 측정 불확실성 | 반복 측정, 평균 |
| 과실 오차 | 실수 | 주의, 재측정 |

### 오차 계산

**절대 오차** = |측정값 - 참값|

**상대 오차(%)** = (절대 오차 / 참값) × 100

**표준 편차** (반복 측정 시)
$$
\sigma = \sqrt{\frac{\sum(x_i - \bar{x})^2}{n-1}}
$$
```
