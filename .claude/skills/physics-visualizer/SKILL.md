# Physics Visualizer: 물리 시각화 도구 생성 스킬

## 역할

물리 개념을 직관적으로 이해할 수 있는 인터랙티브 시각화 도구를 생성합니다.
교사가 수업에서 바로 사용하거나, 학생들이 직접 조작해볼 수 있는 형태로 제공합니다.

## 시뮬레이션 라이브러리

### 역학

#### 포물선 운동 (Projectile Motion)
```
조작 변수:
- 초기 속력 (v₀)
- 발사 각도 (θ)
- 중력 가속도 (g)

출력:
- 실시간 궤적
- 최고점, 수평도달거리
- v-t, y-t 그래프 동기화
```

#### 단진자 (Simple Pendulum)
```
조작 변수:
- 진자 길이 (L)
- 초기 각도 (θ₀)
- 중력 가속도 (g)

출력:
- 진동 애니메이션
- 주기 측정
- 에너지 막대그래프
```

#### 충돌 (Collision)
```
조작 변수:
- 질량 (m₁, m₂)
- 초기 속도 (v₁, v₂)
- 탄성계수 (e)

출력:
- 충돌 전후 운동
- 운동량 보존 확인
- 에너지 변화 표시
```

### 파동

#### 파동 중첩 (Wave Superposition)
```
조작 변수:
- 파장 (λ₁, λ₂)
- 진폭 (A₁, A₂)
- 위상차 (Δφ)

출력:
- 개별 파동 + 합성파
- 보강/상쇄 간섭 표시
```

#### 정상파 (Standing Wave)
```
조작 변수:
- 현의 길이 (L)
- 장력 (T)
- 진동수 (f)

출력:
- 마디/배 위치 표시
- n차 조화진동
```

### 전자기

#### 전기장 시각화
```
조작 변수:
- 전하량 (+q, -q)
- 전하 위치

출력:
- 전기력선
- 등전위선
- 임의 점 전기장 벡터
```

#### 전자기 유도
```
조작 변수:
- 자기장 세기 (B)
- 코일 면적 (A)
- 회전 속도 (ω)

출력:
- 유도 기전력 그래프
- 자속 변화 시각화
```

## 교육적 가이드라인

### 시뮬레이션 설계 원칙

```
1. 단순성: 핵심 개념에 집중, 불필요한 요소 제거
2. 조작성: 학생이 변수를 직접 조절 가능
3. 시각적 피드백: 변화를 즉시 확인 가능
4. 현실성: 실제 물리 법칙에 부합
5. 안내성: 관찰 포인트, 질문 제공
```
