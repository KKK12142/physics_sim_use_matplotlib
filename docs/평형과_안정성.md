## 1. 평형이란?

### 정의
**평형(Equilibrium)**이란 물체에 작용하는 모든 힘과 돌림힘(토크)의 합이 0이 되어, 물체의 운동 상태가 변하지 않는 상태를 말합니다.

### 평형의 두 가지 측면

| 구분 | 정의 | 조건 |
|------|------|------|
| **병진 평형** | 물체가 직선 운동 상태를 유지 | 알짜힘 = 0 |
| **회전 평형** | 물체가 회전 상태를 유지 | 알짜 돌림힘 = 0 |

> **참고**: 평형 상태의 물체는 정지해 있거나 등속 직선 운동을 합니다.

---

## 2. 평형의 조건

### 제1조건: 힘의 평형 (병진 평형)

물체에 작용하는 모든 힘의 벡터 합이 0이어야 합니다.

$$\sum \vec{F} = 0$$

성분별로 나타내면:
$$\sum F_x = 0 \quad \text{(수평 방향)}$$
$$\sum F_y = 0 \quad \text{(수직 방향)}$$

#### 예시: 천장에 매달린 물체
```
        T (장력)
        ↑
        |
        ●  ← 물체
        |
        ↓
        W (무게)
```
평형 조건: $T = W = mg$

### 제2조건: 돌림힘의 평형 (회전 평형)

물체에 작용하는 모든 돌림힘의 합이 0이어야 합니다.

$$\sum \tau = 0$$

#### 돌림힘(토크)의 정의
$$\tau = r \times F = rF\sin\theta$$

- $r$: 회전축에서 힘의 작용점까지의 거리
- $F$: 작용하는 힘의 크기
- $\theta$: 힘과 위치 벡터 사이의 각도

#### 예시: 시소의 평형
```
    ←── d₁ ──→←─── d₂ ───→
         m₁●         ●m₂
    ════════▲════════════
           받침점
```
평형 조건: $m_1 g d_1 = m_2 g d_2$

## 3. 평형의 종류

물체가 평형 위치에서 약간 벗어났을 때 어떻게 반응하는지에 따라 평형을 세 가지로 분류합니다.

### 3.1 안정 평형 (Stable Equilibrium)

**정의**: 물체를 평형 위치에서 약간 벗어나게 하면, **원래 위치로 돌아오려는** 복원력이 작용하는 평형

**특징**:
- 퍼텐셜 에너지가 **최솟값**인 위치
- 외부 힘이 제거되면 원래 위치로 복귀

**예시**:
```
     ⌒⌒⌒⌒⌒⌒⌒
    /         \
   /    ●      \    ← 구슬이 오목한 그릇 바닥에 있는 경우
  ────────────────
```

- 그릇 바닥에 놓인 구슬
- 진자의 최저점
- 계곡 바닥에 놓인 공

### 3.2 불안정 평형 (Unstable Equilibrium)

**정의**: 물체를 평형 위치에서 약간 벗어나게 하면, **원래 위치에서 더 멀어지는** 방향으로 힘이 작용하는 평형

**특징**:
- 퍼텐셜 에너지가 **최댓값**인 위치
- 작은 교란에도 평형이 깨짐

**예시**:
```
        ●        ← 볼록한 면 위의 구슬
    ⌣⌣⌣⌣⌣⌣⌣
  ────────────────
```

- 볼록한 면 꼭대기에 놓인 구슬
- 연필을 뾰족한 끝으로 세워 놓은 경우
- 언덕 꼭대기에 놓인 공

### 3.3 중립 평형 (Neutral Equilibrium)

**정의**: 물체를 평형 위치에서 벗어나게 해도, 새로운 위치에서 **그대로 평형을 유지**하는 상태

**특징**:
- 퍼텐셜 에너지가 **일정**한 위치
- 어느 위치에서나 평형

**예시**:
```
    ●  →  ●  →  ●    ← 수평면 위의 구슬
  ══════════════════
```

- 수평면 위에 놓인 공
- 마찰 없는 수평면 위의 물체

### 평형 종류 비교 (퍼텐셜 에너지 그래프)

```
    E                         E                         E
    |     /\                  |                         |
    |    /  \                 |    ●                    | ●────────●
    |   /    \                |   /\                    |
    |  / ●    \               |  /  \                   |
    | /        \              | /    \                  |
    ──────────── x            ──────────── x            ──────────── x

    안정 평형                  불안정 평형                중립 평형
    (최솟값)                   (최댓값)                   (일정)
```

## 4. 무게중심과 안정성

### 4.1 무게중심 (Center of Gravity)

**정의**: 물체의 전체 무게가 작용한다고 볼 수 있는 한 점

균일한 중력장에서 무게중심은 질량중심과 같습니다.

$$\vec{r}_{cm} = \frac{\sum m_i \vec{r}_i}{\sum m_i}$$

### 4.2 안정성을 결정하는 요소

#### 1) 무게중심의 높이
- **무게중심이 낮을수록** 안정함
- 예: 스포츠카 vs 버스

```
    높은 무게중심         낮은 무게중심
    (불안정)             (안정)

       ■
       ■                   ■■■
      ■■■                  ■■■
    ───────              ───────
```

#### 2) 받침면(기저면)의 넓이
- **받침면이 넓을수록** 안정함
- 예: 피라미드 구조

```
    좁은 받침면           넓은 받침면
    (불안정)             (안정)

       ▲                   ▲
      ─ ─               ─────────
```

#### 3) 무게중심의 수직선 위치
- 무게중심에서 내린 수직선이 **받침면 안에 있어야** 평형 유지

```
    안정                 쓰러짐 직전            쓰러짐

      ●                    ●                      ●
      │                     \                      \
      │                      \                      \
    ▓▓▓▓▓                  ▓▓▓▓▓                   ▓▓▓▓▓
    ─────                  ─────                   ─────

  (수직선이              (수직선이               (수직선이
   받침면 안)             받침면 끝)              받침면 밖)
```

### 4.3 안정도(Degree of Stability)

물체가 넘어지기 시작할 때까지 기울일 수 있는 **최대 각도**로 안정도를 측정할 수 있습니다.

$$\tan\theta_{max} = \frac{b}{h}$$

- $b$: 받침면 가장자리에서 무게중심까지의 수평 거리
- $h$: 무게중심의 높이

## 5. 실생활 속 평형과 안정성

### 5.1 건축과 구조물

| 구조물 | 안정성 원리 |
|--------|-------------|
| 피라미드 | 넓은 받침면, 낮은 무게중심 |
| 에펠탑 | 넓은 기저면, 격자 구조로 바람 저항 감소 |
| 고층 빌딩 | 깊은 기초, 내진 설계 |

### 5.2 교통수단

- **자동차**: 무게중심을 낮추기 위해 엔진을 아래쪽에 배치
- **배**: 용골(keel)을 사용하여 무게중심 하강
- **비행기**: 꼬리 날개로 회전 평형 유지

### 5.3 스포츠

- **스모 선수**: 다리를 벌려 받침면 확대
- **서핑**: 무릎을 굽혀 무게중심 하강
- **체조 착지**: 양팔을 벌려 균형 유지

### 5.4 일상생활

```
예시: 쇼핑카트

    무거운 물건을 아래에!

    ┌─────────┐        ┌─────────┐
    │  가벼움  │        │  무거움  │   ← 불안정!
    │─────────│        │─────────│
    │  무거움  │        │  가벼움  │
    └────○○───┘        └────○○───┘
       안정              불안정
```

## 6. 연습 문제

### 기본 문제

**문제 1**: 다음 중 안정 평형에 해당하는 것을 모두 고르시오.
- (가) 테이블 위에 놓인 책
- (나) 바늘 끝에 세운 달걀
- (다) 그네의 최저점
- (라) 언덕 꼭대기의 공

<details>
<summary>정답 보기</summary>

**(가), (다)**
- (가) 테이블 위의 책: 기울어지면 원래 위치로 돌아오려 함 → 안정 평형
- (나) 바늘 끝의 달걀: 조금만 기울어져도 떨어짐 → 불안정 평형
- (다) 그네의 최저점: 퍼텐셜 에너지 최솟값 → 안정 평형
- (라) 언덕 꼭대기의 공: 굴러 내려감 → 불안정 평형

</details>

---

**문제 2**: 길이 4m, 질량 20kg인 균일한 막대가 수평으로 놓여 있다. 왼쪽 끝에서 1m 지점을 받침점으로 할 때, 오른쪽 끝에 몇 kg의 추를 달아야 평형을 이루는가?

```
         1m              3m
    ├────────┼────────────────────┤
    A        ▲                    B
           받침점                 m=?

    막대 질량: 20kg (중심에 작용)
```

<details>
<summary>풀이 보기</summary>

**풀이**:
막대의 무게중심은 중앙(왼쪽 끝에서 2m)에 있습니다.

받침점을 기준으로 돌림힘의 평형:
- 막대의 무게에 의한 돌림힘: $20g \times 1m$ (시계 방향)
- 추의 무게에 의한 돌림힘: $m \times g \times 3m$ (반시계 방향)

평형 조건:
$$20g \times 1 = m \times g \times 3$$
$$m = \frac{20}{3} \approx 6.67 \text{ kg}$$

**답: 약 6.67 kg**

</details>

**문제 3**: 높이가 60cm이고 밑면의 지름이 40cm인 원기둥이 있다. 이 원기둥이 넘어지지 않고 기울일 수 있는 최대 각도는?

<details>
<summary>풀이 보기</summary>

**풀이**:
균일한 원기둥의 무게중심은 기하학적 중심(높이의 1/2)에 있습니다.

- 무게중심 높이 $h = 30$ cm
- 받침면 가장자리에서 무게중심까지의 수평 거리 $b = 20$ cm (반지름)

최대 기울임 각도:
$$\tan\theta_{max} = \frac{b}{h} = \frac{20}{30} = \frac{2}{3}$$
$$\theta_{max} = \arctan\left(\frac{2}{3}\right) \approx 33.7°$$

**답: 약 33.7°**

</details>

### 심화 문제

**문제 4**: 경사각이 θ인 빗면 위에 밑면의 한 변이 a이고 높이가 h인 정육면체가 놓여 있다. 이 정육면체가 미끄러지기 전에 먼저 넘어지기 위한 조건을 구하시오. (정지 마찰 계수: μ)

<details>
<summary>풀이 보기</summary>

**풀이**:

**넘어지는 조건**: 무게중심에서 내린 수직선이 받침면 밖으로 나갈 때
$$\tan\theta > \frac{a}{h}$$

**미끄러지는 조건**: 경사면 방향의 힘이 최대 정지 마찰력보다 클 때
$$\tan\theta > \mu$$

**넘어지기 전에 먼저 미끄러지려면**:
$$\frac{a}{h} < \mu$$

**미끄러지기 전에 먼저 넘어지려면**:
$$\frac{a}{h} > \mu$$

즉, $a > \mu h$

**답**: $a > \mu h$ (또는 $\frac{a}{h} > \mu$)

</details>

## 핵심 정리

| 개념 | 핵심 내용 |
|------|-----------|
| 평형 조건 | $\sum F = 0$, $\sum \tau = 0$ |
| 안정 평형 | 복원력 작용, 퍼텐셜 에너지 최솟값 |
| 불안정 평형 | 평형에서 멀어짐, 퍼텐셜 에너지 최댓값 |
| 중립 평형 | 새 위치에서 평형 유지, 퍼텐셜 에너지 일정 |
| 안정성 증가 | 무게중심 낮추기, 받침면 넓히기 |
